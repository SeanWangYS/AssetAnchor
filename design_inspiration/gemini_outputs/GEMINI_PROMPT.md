# Gemini 圖片生成提示詞 — 投資組合追蹤 App 畫面

> **使用方式**：把下面〈共用設定〉先丟給 Gemini（建議 Gemini 2.5 Flash Image / Imagen 4），
> 然後再貼〈畫面 1〉、〈畫面 2〉…逐一生成。
> **強烈建議一次一張**，不要叫它一次出全部 — Gemini 出多張時細節會劣化。
>
> Gemini 圖像模型版本：`gemini-2.5-flash-image-preview` 或 Google AI Studio 的 Imagen。

---

## 共用設定（每次對話一開始貼一次）

```
你是一位資深 iOS / Android 行動應用 UI 設計師，正在為一款「個人投資組合追蹤 App」
（產品名稱：Portfolio Tracker，台灣使用者導向）繪製高保真設計稿（high-fidelity mockup）。

【產品定位】
- 跨多券商追蹤股票/ETF 持倉的 mobile app
- 同時管理台股（TWD）與美股（USD）
- 使用者為小資族 ETF + 個股投資者
- 平台：iOS（iPhone 14/15 size, 393 × 852 pt），但版面也適用於 Android
- 框架：React Native + Expo（設計需可在 React Native 實作）

【整體視覺風格】
- **Dark-first**：主色背景使用深沉的近黑色 #0E1116
- **強調色**：金融科技藍綠 #00D6A1（漲）與沉穩紅 #FF5C5C（跌）
- 次要色：薄霧灰 #1A1F26（卡片背景）、霧灰文字 #8A92A3
- 主文字：純白 #FFFFFF；輔助文字：#8A92A3
- **字體**：iOS 系統字 SF Pro Display（標題），SF Pro Text（內文）；數字用 SF Pro Rounded
- **圓角**：卡片 16pt、按鈕 12pt、頭像/logo 圓 28pt
- **間距**：頁面左右 padding 20pt；卡片內 padding 16pt
- **狀態列**：保留 iPhone Dynamic Island，時間顯示 9:41
- **底部 Tab Bar**（4 個 tab，固定）：
  1. 📊 持倉（Holdings）
  2. 📝 交易（Transactions）
  3. 🏦 帳戶（Accounts）
  4. ⚙️ 設定（Settings）

【數字呈現規則】
- 大額用千分位逗號分隔（NT$1,238,540、$8,153.31）
- 漲跌一律用 ▲ / ▼ 開頭，後接金額與百分比
- 台股用「NT$」開頭、美股用「US$」或「$」開頭
- 不要用奇怪的等寬字體，數字維持優雅圓潤

【語言】
- 所有畫面文字使用**繁體中文**（台灣用語）
- 股票代號保留原文（例：2330、AAPL、0050、VTI）
- 股票名稱可中英並陳：「台積電 2330」「Apple AAPL」

【需要避免的東西】
- 不要出現「Trade / 交易」這種會誤導為下單功能的主按鈕（本 App 不能下單，只追蹤）
- 不要出現幣別兌換或加密貨幣相關圖示（MVP 不做）
- 不要出現 K 線圖（candlestick）— MVP 只做簡單線圖
- 不要圓環式資產配置圖（pie / donut chart）— 第三階段才做

收到請回覆「了解，準備好接收第一張畫面的指示」。
```

## 進階指令小抄

### 想叫 Gemini 補強某個區塊時可加上：
- **想要更精細**：「請把這張圖的線圖部分放大到佔畫面 35%，曲線細節更平滑、加上垂直 hover 線」
- **想要實機框**：「請把整張畫面放進 iPhone 15 Pro 的太空灰機身框內，背景使用淡灰 #F7F7F7」
- **想要 light mode 對照**：「以同樣版面，但改成 light mode：白底 + 純黑文字 + 漲跌色不變」
- **想要 2 連畫面**：「請輸出兩張並排（總覽頁 + 個股詳情頁），中間有 24px 間距，整體背景透明」

### 如果 Gemini 出來文字亂碼：
中文字在 Imagen 系列模型仍有不穩定狀況，可以加：
- 「**文字部分請用英文**：『Holdings』『Total Assets』『Add Transaction』」，視覺確認後再人工 PS 換成中文，或直接把產出圖丟到 Figma 作為 wireframe 參考。

## 注意事項

1. **Gemini 圖片生成模型對精確文字仍不可靠**，產生的內容只能當「視覺氛圍稿」，
   不能直接當 Figma final 用。先用它對齊**版面骨架與色彩風格**就好。
2. **產出後存到 `design_inspiration/gemini_outputs/`**，
   等你下次 chat 我可以幫你看哪張最接近、需要怎麼微調 prompt 再生一次。
3. 想看更深入真實 App user flow，**Mobbin** 訂閱比 Gemini 生成有用十倍 — 它直接給你 Robinhood / Revolut / Public 的全部畫面截圖。
