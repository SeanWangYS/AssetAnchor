## B. 分析頁面內含圖表（使用者構想）

頁面會用多張圖表，從不同維度看「現有資產」：

### 1. 資產配置圓餅圖（Portfolio Allocation）
- 圓餅 / 圓環圖。
- 每個標的佔**總資產**的比例分布。
- 用途：一眼看出持倉集中度 / 分散程度。

### 2. 各標的「當前總市值 vs 投入總成本」對比柱狀圖
- 橫向把**所有標的**放在一起比較。
- **重點**：同一個標的在**同一根柱子（同一 row）**上同時呈現兩個量：
  - 該資產**當前總絕對價值**（市值）
  - 該資產**投入總成本**
- 形式：每個標的一組（grouped / overlay bar），同列上能直接看出「賺/賠的幅度」（市值 vs 成本的差）。
- 例：TSMC、VOO、QQQ… 各一組，全部排在一起。

### 3. 各標的「總報酬率」比較柱狀圖
- 柱狀圖，把每個標的的**總報酬率（%）**放在一起比較。

### 4. 各標的「年化報酬率」比較柱狀圖
- 柱狀圖，把每個標的的**年化報酬率（%）**放在一起比較。
- （註：年化報酬率在 planning.md 屬第二階段計算，此處為分析頁的呈現需求。）

### 5. 各標的「目前未實現損益」比較柱狀圖
- 柱狀圖，把每個標的的**目前未實現損益（金額）**放在一起比較。
- 正負值（賺/賠）需可區分。
